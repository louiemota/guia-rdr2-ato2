# Guia de Fronteira RDR2

Guia estático em PT-BR para o Capítulo 2 e começo do Capítulo 3 de Red Dead Redemption 2.

- Rota de desbloqueios por missão
- Localizações úteis descritas pelo mapa
- Receitas corretas das bolsas do Pearson
- Caça, armas, conteúdo perdível e preparação para o Capítulo 3
- Spoilers leves ocultos e checklist salvo em `localStorage`

Abra `index.html` no navegador. O projeto continua compatível com GitHub Pages.
